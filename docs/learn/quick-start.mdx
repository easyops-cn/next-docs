## Responding to events

```yaml
brick: button
properties:
  textContent: Click me
// highlight-start
events:
  click:
    action: console.log
    args:
    - You clicked me
// highlight-end
```

## Displaying data

```yaml
brick: div
properties:
// highlight-next-line
  textContent: <% CTX.user.name %>
```

## Conditional rendering

```yaml
- if: <% CTX.isAdmin %>
  brick: admin-panel
- if: <% !CTX.isAdmin %>
  brick: user-panel
```

```yaml
brick: ':if'
dataSource: <% FN.check('my', 'complex', 'expression') %>
children:
- brick: admin-panel-a
- brick: admin-panel-b
- brick: user-panel-x
  slot: else
- brick: user-panel-y
  slot: else
```
